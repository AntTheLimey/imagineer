## Quick Reference

### Key Style Rules

- Active voice throughout
- 79-character line wrapping
- Blank line before lists
- Complete sentences in bullets
- Code blocks with language tags

### File Organization

- Documentation in `/docs/`
- Lowercase filenames
- CHANGELOG.md for changes
- Todo.md for tracking
